# Js-lesssn-3-1
Js Lesson 3_1 (Bolean type)


```node
 5 > 3     // true
 5 < 3     // false
 5 <= 5    // true
 5 >= 3    // true

 5 == 5    // true
 5 == '5'  // true   !!!
 5 === '5' // false  !!!

 5 > '3'     // true
 5 < '3'     // false
 5 <= '5'    // true
 5 >= '3'    // true

 5 != 5      // false
 5 == 5      // true
 5 != '5'    // false  !!!
 5 !== '5'   // true   !!!
 ```
